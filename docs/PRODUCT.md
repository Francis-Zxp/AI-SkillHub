# SkillHub Product Context

## Users

Researchers, graduate students, and lab members who collect AI Skills from GitHub and want one shared folder that works across Claude Code, Codex, Antigravity, and similar agents.

## Product Purpose

SkillHub turns scattered GitHub Skill downloads into a manageable local hub. It clones or updates source repositories, scans for real `SKILL.md` folders, links active Skills into one shared directory, removes stale managed links, and gives users visible success or failure feedback.

## Register

product

## Principles

- Keep the active `skills` directory clean and understandable.
- Prefer reversible actions: move old copies to archives instead of deleting them.
- Make path portability explicit because users may copy the folder to a different drive, name, or computer.
- Show clear status after every action: success, failure, and next step.
- Keep the interface calm, practical, and friendly for non-programmers.

## Anti-References

- No developer-only dashboards full of unexplained logs.
- No silent background work without a visible report.
- No hidden destructive cleanup.
- No assumption that every user has the same folder name or drive letter.
