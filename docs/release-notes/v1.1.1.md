# AI SkillHub v1.1.1

AI SkillHub v1.1.1 is a sharing-stability update. It is focused on first-run guidance, clearer AI tool states, troubleshooting bundles, and safer release packaging.

## Highlights

- Adds a first-run guide and Quick Start help dialog.
- Explains Claude Code, Codex, and Antigravity detection states more clearly.
- Treats missing optional AI tools as readable information instead of a program failure.
- Exports sanitized troubleshooting bundles for another computer's failure report.
- Adds clean share-recipient validation with `AI SkillHub.exe --share-recipient-test`.
- Adds release package preflight with `AI SkillHub.exe --release-preflight`.

## Release Preflight

Before publishing this version, run:

```text
AI SkillHub.exe --release-preflight
```

The preflight creates `release/AI-SkillHub-v1.1.1.zip`, writes a SHA256 file, and checks that the package does not contain personal Skills, cloned GitHub sources, local config, reports, WebView2 cache, or private reference material.

## Chinese Notes
