# AI SkillHub v1.1.0

AI SkillHub v1.1.0 is the first public sharing release.

## Highlights

- Manage a shared Skills folder for AI coding tools.
- Add GitHub Skill repositories and sync them locally.
- Import local folders or zip files after previewing whether they contain `SKILL.md`.
- Detect Claude Code, Codex, and Antigravity without creating fake tool folders.
- Export diagnostics when another computer cannot run or sync correctly.
- Keep personal Skills, cloned sources, reports, caches, and local config out of the public repository.

## Download

Download `AI-SkillHub-v1.1.0.zip` from the GitHub Release assets.

## First Launch

1. Unzip the package.
2. Run `AI SkillHub.exe`.
3. Add a GitHub Skill repository or import a local Skill folder/zip.
4. Click sync.
5. If no AI coding tool is detected, install or initialize Claude Code, Codex, or Antigravity first.
